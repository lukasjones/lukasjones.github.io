What is SQL you ask?

SQL stands for Structure Query Language. It's used to ask and tell the database to show and do stuff.

Bellow are a list of cheats (query's(commands)) and their outputs. Commands/querys/cheats are written in upper case. This is not required when querying the database but is done a lot to differentiate commands from other pieces of the statemtent.

  • SELECT and FROM

    SELECT is used in the majority of querys to ask for information FROM a table. Alternatively, it can also be used to SELECT and change something in a table(more on that later).

    SELECT * FROM table_name;
    // This code selects all the tables from the specified table_name.
    SELECT column_name_1, column_name_2 FROM table_name;
    // You can select specific columns by using the name of the columns.
    SELECT * FROM grocery_list;
    // Ouput:
    id list_name total_cost total_weight completed?
    1 Safeway 56.98 20 no
    2 Trader Joes 25.00 7 no
    3 Whole Foods 299.99 56 yes
    4 Costco 200.00 50 no

    SELECT id, list_name FROM grocery_list;
    // Ouput:
    id list_name
    1 Safeway
    2 Trader Joes
    3 Whole Foods
    4 Costco

  • WHERE

    WHERE is used to query a specific row in a table.

    SELECT * FROM table_name WHERE column_name (operator) value;
    // This will select all the columns where a specific column_name is a specific value. Basically it gives you the row or rows where that column_name = the value.
    // For example.
    SELECT * FROM grocery_list WHERE id = 3;
    // Output:
    id list_name total_cost total_weight completed?
    3 Whole Foods 299.99 56 yes

  • AND & OR

    AND and OR are used in SQL in conjunction with WHERE to limit or open the search. It's just like the Ruby && and ||.

    SELECT * FROM table_name WHERE column_name_1 = value AND column_name_2 = value;
    // This will select all the columns where a column_name_1 = the value specified AND column_name_2 = the value specified. If they are not both true, the row will not be selected.
    SELECT * FROM grocery_list WHERE ID = 3 AND completed = 'yes';
    // Output:
    id list_name total_cost total_weight completed?
    3 Whole Foods 299.99 56 yes

    SELECT * FROM table_name WHERE column_name_1 = value OR column_name_2 = value;
    // This will select all the columns where a column_name_1 = the value or column_name_2 = the value. Basically it's selecting the rows that return true for both the WHERE statements.
    SELECT * FROM grocery_list WHERE id = 2 OR id = 4;
    // Output:
    id list_name total_cost total_weight completed?
    2 Trader Joes 25.00 7 no
    4 Costco 200.00 50 no

  • ORDER BY

    ORDER BY is used to order your output from a SELECT statement. You can order items in ASCending and DESCending
